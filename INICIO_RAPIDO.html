Inicio Rápido — Programación Python Fundamentos

5 minutos para tener todo listo y empezar a aprender.

¿Qué es este curso?

Un curso completo de Python para principiantes con 70+ desafíos prácticos, 10 laboratorios, 12 quizzes y 5 proyectos integradores. Inspirado en MIT 6.0001, diseñado para ABACOM.

Paso 1: Instala Python (2 min)

BASH
# macOS (Homebrew)
brew install python

# Linux (Ubuntu/Debian)
sudo apt update && sudo apt install python3 python3-pip

# Windows: descarga desde python.org (NO Microsoft Store)
# https://www.python.org/downloads/

Verifica:

BASH
python3 --version
# Debe mostrar Python 3.9 o superior

Paso 2: Instala VS Code (1 min)

  1. Descarga desde code.visualstudio.com
  2. Abre VS Code → Extensions (Ctrl+Shift+X) → Busca “Python” → Install
  3. Instala la extensión de Microsoft (la oficial)

Paso 3: Clona el Curso (1 min)

BASH
git clone https://github.com/statick88/Course_of_python.git
cd Course_of_python

Paso 4: Abre y Explora (1 min)

BASH
# Abre en VS Code
code .

# O abre el curso en tu navegador
open docs/index.html        # macOS
xdg-open docs/index.html    # Linux
start docs/index.html       # Windows

Tu Primer Desafío (30 segundos)

Abre content/modulo-01/desafio-01-saludo-personalizado.qmd y lee la descripción. Luego:

  1. Crea un archivo hola.py
  2. Escribe: print("¡Hola, mi nombre es TU_NOMBRE!")
  3. Ejecuta: python3 hola.py

¡Felicidades! Acabas de empezar el curso. 🎉
