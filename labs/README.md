## 🔧 Troubleshooting Común

### Problema: Python no se encuentra
**Soluciones:**
1. Verificar instalación: `python3 --version`
2. En Windows: `python --version`
3. Reinstalar desde python.org si es necesario

### Problema: Error de sintaxis
**Soluciones:**
1. Verificar indentación (4 espacios consistente)
2. Cerrar paréntesis y comillas
3. Usar linter en VS Code (extensión Python)

### Problema: Módulo no encontrado
**Soluciones:**
1. Verificar que archivo está en mismo directorio
2. Usar `import sys; print(sys.path)` para ver rutas
3. Verificar nombre exacto del módulo
