# Tarea 1: Configuración del Entorno de Desarrollo

**Módulo de referencia**: Módulo 1 - Introducción a Python
**Unidad**: 1.2 Instalación
**Dificultad**: ⭐ Fácil
**Tiempo estimado**: 30 minutos

## Descripción

Configura tu entorno de desarrollo local para trabajar con Python. Esta tarea es fundamental para todo el curso.

## Pasos

1. **Instala Python 3.9 o superior**
   - Descarga desde [python.org](https://python.org)
   - Verifica con `python3 --version`

2. **Instala VS Code**
   - Descarga desde [code.visualstudio.com](https://code.visualstudio.com)
   - Instala la extensión "Python" de Microsoft

3. **Crea tu primer proyecto**
   - Crea una carpeta `mi-primer-programa`
   - Dentro, crea un archivo `main.py`
   - Escribe: `print("¡Hola, Python!")`
   - Ejecuta con `python3 main.py`

4. **Verifica tu instalación**
   - Ejecuta `python3 -c "import sys; print(sys.version)"`
   - Debe mostrar Python 3.9+

## Criterios de Éxito

- [ ] Python 3.9+ instalado y funcionando
- [ ] VS Code con extensión Python instalada
- [ ] Primer programa ejecutado exitosamente
- [ ] Puedes abrir la terminal integrada en VS Code
