# Tareas 6-10: Proyectos Intermedios y Finales

## Tarea 6: Agenda de Contactos con Diccionarios (Semana 6)

**Módulo**: 6 - Diccionarios y Sets
**Dificultad**: ⭐⭐ Intermedio

Crea una agenda de contactos donde cada contacto es un diccionario con nombre,
teléfono, email y dirección. Debe permitir buscar, agregar, editar y eliminar
contactos.

**Entrega**: `agenda-contactos.py`

---

## Tarea 7: Diario Personal con Archivos (Semana 7)

**Módulo**: 7 - Manejo de Archivos
**Dificultad**: ⭐⭐ Intermedio

Construye un diario personal que guarde entradas con fecha y hora en un archivo
de texto. Debe permitir escribir nuevas entradas y leer todas las anteriores.

**Entrega**: `diario-personal.py`

---

## Tarea 8: API del Clima con Módulos Externos (Semana 8)

**Módulo**: 8 - Módulos y Paquetes
**Dificultad**: ⭐⭐⭐ Avanzado

Usa el módulo `requests` para consumir una API del clima (OpenWeatherMap). El
usuario ingresa una ciudad y se muestra el clima actual con temperatura, humedad
y descripción.

**Entrega**: `api-clima.py`

---

## Tarea 9: Refactorización de Código (Semana 9)

**Módulo**: 9 - Buenas Prácticas
**Dificultad**: ⭐⭐⭐ Avanzado

Te proporcionamos un archivo de código "sucio" (sin docstrings, nombres
crípticos, sin manejo de errores). Tu misión es refactorizarlo siguiendo PEP 8,
con nombres descriptivos, docstrings y manejo de errores.

**Entrega**: Código original + refactorizado con explicación de cambios

---

## Tarea 10: Proyecto Integrador Final (Semana 10-12)

**Módulo**: 10 - Proyecto Integrador
**Dificultad**: ⭐⭐⭐⭐ Integrador
**Tiempo**: 4-6 horas

Elige UNO de los siguientes proyectos:

### Opción A: Gestor de Tareas Completo
- CRUD de tareas con prioridades, fechas y estados
- Persistencia en JSON
- Estadísticas y reportes
- Menú interactivo

### Opción B: Juego del Ahorcado (MIT-inspired)
- Carga palabras desde archivo
- Tracking de letras adivinadas
- Límite de intentos
- Visualización ASCII del ahorcado

### Opción C: Cifrado César y Criptoanálisis (MIT-inspired)
- Funciones de encriptar/desencriptar
- Fuerza bruta para crackear mensajes
- Detección automática de texto válido

### Opción D: Analizador de Datos CSV
- Lectura de archivos CSV
- Estadísticas (media, mediana, moda)
- Filtrado y agrupación
- Reporte en texto formateado

**Entrega**:
- [ ] Código fuente completo
- [ ] README con instrucciones
- [ ] Tests básicos
- [ ] Capturas de ejecución
- [ ] Reflexión de 1 párrafo

**Fecha límite**: Semana 12
