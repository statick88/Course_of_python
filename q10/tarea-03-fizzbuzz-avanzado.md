# Tarea 3: FizzBuzz Avanzado

**Módulo de referencia**: Módulo 3 - Estructuras de Control
**Unidad**: 3.3 For Loop
**Dificultad**: ⭐⭐ Intermedio
**Tiempo estimado**: 60 minutos

## Descripción

Implementa el clásico FizzBuzz pero con reglas personalizadas. Este ejercicio evalúa tu dominio de condicionales y bucles.

## Requisitos

1. El usuario ingresa un número límite (ej: 50)
2. El programa recorre números del 1 al límite
3. Para cada número:
   - Si es divisible por 3 → imprime "Fizz"
   - Si es divisible por 5 → imprime "Buzz"
   - Si es divisible por 3 Y 5 → imprime "FizzBuzz"
   - Si no → imprime el número
4. Al final, muestra estadísticas:
   - Cuántos "Fizz" hubo
   - Cuántos "Buzz" hubo
   - Cuántos "FizzBuzz" hubo
   - Cuántos números normales

## Criterios de Éxito
