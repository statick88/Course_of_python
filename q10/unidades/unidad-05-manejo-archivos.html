🎯 Objetivo: Leer y escribir archivos de texto, CSV y JSON. Usar open() con contexto with, manejar rutas con pathlib y aplicar manejo de errores en I/O.
🎮 Objetivos de Prueba:

  • Real Python: Working With JSON Data (Tutorial práctico).

// Instrucción: Leer CSV y convertir a diccionario con persistencia JSON

import csv, json
with open("datos.csv") as f:
    lector = csv.DictReader(f)
    datos = [dict(fila) for fila in lector]
with open("salida.json", "w") as f:
    json.dump(datos, f, indent=2, ensure_ascii=False)
