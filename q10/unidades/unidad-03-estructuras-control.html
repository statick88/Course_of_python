// Instrucción: Implementar FizzBuzz con comprensión de listas

resultado = [
    "FizzBuzz" if i % 15 == 0
    else "Fizz" if i % 3 == 0
    else "Buzz" if i % 5 == 0
    else str(i)
    for i in range(1, 51)
]
⚖️ Rúbrica de Evaluación:
  • 40% Condicionales: Ramas lógicas correctas con operadores de comparación.
  • 30% Bucles: Uso apropiado de for y while según el escenario.
  • 30% Comprensión: Filtrar y transformar listas con sintaxis compacta.
